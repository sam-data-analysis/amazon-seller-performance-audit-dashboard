# Amazon Seller Performance Audit Dashboard
### A Power BI dashboard analysing Amazon seller performance, identifying wasted ad spend and scaling opportunities.

I built this project to analyse e-commerce and Amazon-style advertising data, with the goal of identifying wasted spend, uncovering scaling opportunities, and improving overall profitability.

This dashboard simulates a real-world client audit, focusing on turning data into clear commercial actions rather than just reporting metrics.

## Project Overview

In this project, I analysed SKU-level revenue, profit, and advertising performance to answer key business questions:

- Which products are generating revenue but losing money?
- Where is advertising spend being wasted?
- Which SKUs deliver strong returns and should be scaled?
- How can budget be reallocated to improve profitability?

The focus was to replicate how a data analyst would approach a real client problem.

---

## What I Did

- Built a full SKU-level dataset combining revenue, profit, and advertising metrics  
- Analysed relationships between ad spend, ROAS, and profitability  
- Created a classification framework to group SKUs into:
  - Efficient  
  - Wasteful  
  - Scaling Opportunity  
  - Margin Pressure  
- Designed a Power BI dashboard to communicate insights clearly to non-technical stakeholders  
- Translated analysis into actionable business recommendations  

---

## Key Insights

- A large share of advertising spend is allocated to low-ROAS SKUs, reducing overall efficiency  
- Several products generate strong revenue but remain unprofitable, indicating cost or pricing issues  
- High-ROAS products often have relatively low ad spend, highlighting clear opportunities to scale  
- Reallocating spend from inefficient SKUs to stronger performers could materially improve profit
